데이터를 저장할 Storage를 추가한다. <br>
이번실습에서는 Blob만 사용하지만 SQL과 NoSQL에 저장도 가능하다.<br>
리소스 만들기 -> Storage -> Storage 계정…<br>

<p>
이름은 소문자로 입력해야 되며 위치 선택하고 기존 리소스를 선택하고 만들기를 클릭한다.<br>
복제는 LRS로 선택한다.<br>
Azure Storage 복제는 아래 링크를 참조.<br>
